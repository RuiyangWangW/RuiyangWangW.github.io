Research & Publications

LLM-MCoX: Large Language Model Multi-robot Coordinated Exploration and Search

ICRA 2026 (Submitted)

Developed LLM-MCoX, a framework that fuses LiDAR-based frontier clustering and doorway detection with multimodal LLM reasoning to coordinate robot teams. Achieved 22.7% faster exploration and 50% more efficient search than baseline methods, validated with Unitree Go2 quadruped and X500 drone.

Paper Video

Neural Symbolic Deadlock Resolution in Multi-robot Systems

L4DC 2025 (Accepted)

Proposed NSDR combining Active-Passive paradigm with Neural Logic Machines (NLMs) to resolve deadlocks in multi-robot systems. Trained on simple 2-5 robot scenarios and generalized to 8-10+ robots in complex environments, outperforming adaptive repulsive force methods.

Paper Slides

Compatible Constraint Selection in Receding-Horizon Control

ACC 2024 (Accepted)

Developed heuristics for maximal feasible subset selection in receding-horizon control using Lagrange multipliers to identify incompatible soft constraints. Achieved near-optimal performance at orders-of-magnitude lower computation cost than exhaustive methods.

Paper Code

Autonomous Driving Robot (BotLab)

Built a fully autonomous driving robot capable of exploring and navigating in unforeseen mazes. Implemented triple-layered PID controller for wheel speed control, particle filter for localization, and A* path planning algorithm.

C++ PID Control Particle Filter A* Planning
Report

Latent Space Controller for PANDA Arm

Designed a latent space controller for 7-DOF PANDA robotic arm using Variational AutoEncoder (VAE) and Sparse Identification of Nonlinear Dynamics (SINDy). Achieved 20% fewer actions than baseline in planar pushing tasks.

Python VAE SINDy PyBullet
Report

PoseCNN for 6D Object Pose Estimation

Reimplemented PoseCNN network in PyTorch for 6D object pose estimation. Trained on PROPS dataset and improved 5°5cm accuracy from 53.62% to 64.14% by adding feed-forward layers to the baseline network.

PyTorch Computer Vision CNN Pose Estimation
